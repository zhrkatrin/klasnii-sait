Шаблон цього проекту створений для студентів курсу Web Beginner задля виконання завдань по Maker Time.

Проект містить базову структуру папок та файлів, яка використовувалась у навчальних матеріалах протягом року.

У файлі index.html вже є підключення до стилів (style.css) та додатково підключені FontAwesome, для використання текстових іконок.

У теці "Технічне завдання" можна знайти скріншоти двох сторінок сайту, який необхідно створити.
Текстові документи для сайту (щоб не друкувати) можна взяти з самого сайту, посилання на який буде нижче.

Усю необхідну інформацію по створенню веб-сторінок та веб-додатків можна знайти у роздаткових матеріалах.

Номери тем, які можуть бути корисними:

№8-9. Каскадні таблиці стилів
    Заняття, на якому відбувається знайомство з каскадними таблицями стилів, їх підключенням та особливістю написання коду.

№17-18. Flexbox. Вступ
    Вступне заняття по роботі з Flexbox. У роздатці можна згадати, як створити Flexbox контейнер та вирівнювати контент всередині нього.

№19. Адаптивність в Flexbox
    Заняття, у якому розповідається про те, що таке медіа-запити, та як х їх допомогою зробити саййт більш адаптивним.

№22. Grid. Знайомство з сітками
    Вступне заняття по роботі з Grid. У роздатці можна згадати, як створити Grid-контейнер та налаштовувати сітку.
